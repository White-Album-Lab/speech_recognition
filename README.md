## 中文语音识别
##### 1. 环境
- OS: Windows 10 v1903
- Cuda: 10.0
- Python: Miniconda 3.7.3
- Editor: Pycharm Professional 2019.2
- Tensorflow: 2.0
------
##### 2. 训练数据下载
- 清华大学中文语料库（thchs30）[下载地址](http://www.openslr.org/18/)
------
##### 3. 训练
- 配置 `conf` 目录下的 `conf.ini` 文件中的各项
- 运行 `train.py` 开始训练
- 运行 `test.py` 进行测试
- 完全可以直接在cmd或者powershell运行两文件
